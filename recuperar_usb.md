Flashcard técnico y mínimo.

**Instalación gddrescue**

```
sudo apt install gddrescue
```

**Clonado del USB a imagen**

```
sudo ddrescue /dev/sdX usb.img usb.log
```

**Instalación PhotoRec (incluido en testdisk)**

```
sudo apt install testdisk
```

**Ejecución de PhotoRec sobre imagen**

```
sudo photorec usb.img
```

**Ejecución de PhotoRec sobre dispositivo**

```
sudo photorec /dev/sdX
```
